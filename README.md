# Autoportrait vidéo exécutable  
  
  Cet autoportrait explore une dimension plus interactive du portrait vidéo. Il s'affranchit des règles traditionnelles du montage et de la composition vidéographique en compilant différents types d'images tels que des images fixes, des vidéos au format mp4 et des simulations en temps réel. Le pseudo-aléatoire et le génératif jouent un rôle important dans le montage final, qui s'opère en temps réel sur l'appareil du spectateur.

## Installer et lancer le projet  

Ce projet requiert python3 et nécessite d'être exécuté sur une machine compatible.

1. Installer python3 sur ce lien > www.python.org/downloads/  
2. Sur certains systèmes la prochaine étape nécessitera la création d'un environnement virtuel (instructions détaillées dans le fichier venv.sh)
3. Installer les dépendances du projet :
  
  pip install -r dependancies

4. Lancez le projet avec la commande suivante :

  python3 main.py
