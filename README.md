# Autoportrait vidéo exécutable  
  
  Cet autoportrait explore une dimension plus interactive et autrement immersive du portrait vidéo. Il s'affranchit des règles traditionnelles du montage et de la composition vidéographique en compilant différents types d'images tels que des images fixes, des vidéos au format mp4 et des simulations en temps réel. Le pseudo-aléatoire et le génératif jouent un rôle important dans le montage final.

## Installer et lancer le projet  

Ce projet requiert python3 et nécessite d'être exécuté sur une machine compatible.

Installer python3 (TODO lien vers le tutoriel)

Créer un environnement virtuel (Exécuter les commandes dans le fichier venv.sh)

Une fois le venv activé, exécuter la commande suivante :

  
  pip install -r dependancies

cela installera les librairies manquantes nécessaires au bon fonctionnement du projet

lancez le projet avec la commande suivante :

python3 main.py
